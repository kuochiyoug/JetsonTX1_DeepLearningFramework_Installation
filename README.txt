Welcome to Tegra Linux Driver Package documentation. 

This TAR file provides a reference volume containing several documents
for this release. Those include a Development Guide and application notes 
(if any). Follow the steps below to successfully extract and use this 
documentation.

TO USE THIS DOCUMENTATION

(1) Extract this TAR file to your local drive.

    Note: You must fully extract the files in this archive; otherwise,
          links and documents within the archive will not function.

(2) Click to open the Start_L4T_Docs.html file.

    The "Tegra Linux Driver Package Development Guide" window opens in 
    your default browser.

    Once you open the Development Guide you can use its Search tab and
    Navigation pane to look for the information you want. The
    Development Guide also includes a FAQ.
